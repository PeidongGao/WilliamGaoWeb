Journal Harvester SOP provides browser-based workflows and scripts for
extracting bibliographic metadata from publicly accessible journal web pages
and, where applicable, public metadata services such as Crossref.

The included scripts are intended for low-volume, user-initiated execution
through a web browser's developer console. They are not designed as large-scale
crawlers or automated harvesting systems.
